Metodología del Índice de Priorización

El índice combina tres criterios con base en fuentes oficiales. La puntuación máxima posible es 13 puntos.

CriterioPuntajeJustificación
Categoría 1 – 5 pts Jerarquía de nivel de protección según LGEEPA arts. 48–55.
RB=5, PN=4, APFF=3, APRN=2, MN=2, SANT=1
Superficie (log₁₀) 1 – 5 pts Mayor superficie = mayor impacto potencial de conservación.
Sin programa de manejo 0 ó 4 pts Su ausencia indica una brecha de gestión prioritaria para FMCN.
Limitación del índice: no incorpora datos de biodiversidad (CONABIO), ni poblaciones al rededor. En una versión institucional se enriquecería con estas capas para mayor precisión.
